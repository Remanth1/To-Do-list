## üì¶ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Remanth1/To-Do-list.git
   cd To-Do-list
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## üìñ Usage

1. **Register/Login**: Create an account or log in to access your tasks.
2. **Add Tasks**: Use the task input to create new to-do items.
3. **View Tasks**: Switch between different views (Dashboard, Calendar, Mine).
4. **Manage Tasks**: Edit or delete tasks as needed.
5. **Filter Tasks**: Use filter tabs to view specific task categories.
